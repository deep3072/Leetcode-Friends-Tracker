# Leetcode Friends Tracker

A simple Chrome extension to track your friends' LeetCode progress. Easily add friends, view their profiles, recent submissions, and contest performanceâ€”all in one place.

## Features

- **Friend Dashboard**  
  Add LeetCode usernames and view detailed metrics, including rating, problem stats, recent activity, and contest history.

- **In-Page Integration**  
  Injects a tracker button in LeetCode pages for quick access.
